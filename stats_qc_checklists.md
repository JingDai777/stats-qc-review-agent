# Checklist 1 - Oncology interventional studies

| # | Check | Why | Evidence | Checked |
|---:|---|---|---|---|
| 1 | Study design and evidence standard | Ensures conclusions match randomized, single-arm, dose-expansion, basket, or other oncology design limits. | Protocol design section, SAP, registry, manuscript methods. |  |
| 2 | Prespecification of endpoints and subgroup hypotheses | Prevents outcome switching, data-driven subgroup claims, and inflated false-positive risk. | Dated protocol, SAP, registry endpoint list, subgroup plan. |  |
| 3 | Oncology endpoint definitions | OS, PFS, EFS, DFS, ORR, CR, PR, DCR, DoR, and PRO endpoints must be defined consistently. | Protocol endpoint definitions, SAP, TLF shells, manuscript methods. |  |
| 4 | Estimand clarity and intercurrent events | Intercurrent events such as treatment discontinuation, subsequent therapy, crossover, and death affect interpretation. | SAP estimand section and intercurrent-event strategy. |  |
| 5 | Sample size, power, and maturity | Underpowered or immature analyses can overstate benefit or safety conclusions. | Sample-size assumptions, information fraction, event count, follow-up maturity. |  |
| 6 | Multiplicity control | Multiple endpoints, cohorts, subgroups, dose levels, and interim looks inflate type I error. | Testing hierarchy, alpha allocation, gatekeeping, interim plan. |  |
| 7 | Randomization, stratification, and control arm integrity | Randomization and stratification support causal inference and valid adjusted analyses. | Randomization schedule, stratification factors, baseline balance, SAP model. |  |
| 8 | Analysis populations | ITT, mITT, response-evaluable, safety, and biomarker populations affect endpoint interpretation. | Population definitions, disposition TLFs, analysis set flags. |  |
| 9 | Time-to-event endpoints and censoring | PFS/OS estimates are sensitive to censoring, assessment schedule, and follow-up. | Censoring rules, Kaplan-Meier tables, censoring summaries, event listings. |  |
| 10 | Response endpoint evaluation | ORR, CR, PR, DCR, and DoR require correct response criteria and confirmation rules. | RECIST/response criteria, assessment schedule, response TLFs, listings. |  |
| 11 | BICR vs investigator assessment | Discordance between central and investigator assessments can affect endpoint credibility. | BICR charter, adjudication rules, concordance tables, sensitivity analyses. |  |
| 12 | Duration of response and follow-up | DoR interpretation depends on responders, censoring, and maturity. | Responder denominator, DoR KM table, follow-up duration, censoring reasons. |  |
| 13 | Overall survival maturity, crossover, and subsequent therapy | OS can be confounded by crossover, subsequent therapy, and immature follow-up. | OS event count, follow-up, crossover records, subsequent therapy listings. |  |
| 14 | Single-arm limitations and historical control use | Single-arm studies cannot establish comparative efficacy without strong assumptions. | Protocol rationale, historical control method, eligibility and endpoint comparability. |  |
| 15 | Subgroup and biomarker claims | Small or post hoc biomarker/subgroup analyses are unstable and often exploratory. | Prespecified subgroup list, interaction tests, biomarker analysis plan. |  |
| 16 | Missing data, discontinuation, and assessment windows | Informative missingness or missed assessments can bias oncology endpoint estimates. | Missing data summaries, visit windows, discontinuation reasons, sensitivity analyses. |  |
| 17 | Sensitivity and robustness analyses | Alternative censoring, populations, models, and endpoint definitions test conclusion stability. | SAP sensitivity plan and corresponding TLFs. |  |
| 18 | Safety exposure and oncology safety context | AE rates must use correct safety denominator and exposure duration, especially with unequal follow-up. | Safety population, exposure TLFs, TEAE definitions, AESI tables. |  |
| 19 | Cross-trial comparisons and pooling | Naive cross-trial comparisons are confounded by population, design, and follow-up differences. | Indirect comparison methods, NMA/MAIC/STC rationale, heterogeneity assessment. |  |
| 20 | Claim language discipline | Manuscript conclusions must match design, maturity, multiplicity, and uncertainty. | Abstract, results, discussion, limitations, conclusion wording. |  |

# Checklist 2 - Non-oncology interventional studies

| # | Check | Why | Evidence | Checked |
|---:|---|---|---|---|
| 1 | Design-question alignment | Objective, population, intervention, comparator, endpoint, and analysis must answer the same question. | Protocol objectives, endpoint definitions, SAP analysis section. |  |
| 2 | Registration, protocol, and SAP prespecification | Prespecification protects against outcome switching and selective reporting. | Dated protocol/SAP, registry entry, version history. |  |
| 3 | Estimand clarity | Defines the treatment effect of interest and aligns design, conduct, analysis, and interpretation. | Estimand section: population, variable, intercurrent events, summary measure. |  |
| 4 | Endpoint validity and timing | Endpoints should be meaningful, validated, and measured at appropriate visits. | Endpoint definitions, assessment schedule, validation references. |  |
| 5 | Sample size and power | Assumptions should justify primary and key secondary analyses. | Power calculations, attrition assumptions, effect size, event rate. |  |
| 6 | Randomization and allocation concealment | Protects against selection bias and supports valid inference. | Randomization method, blocking, stratification, allocation process. |  |
| 7 | Blinding and outcome assessment | Subjective outcomes need protection from measurement and ascertainment bias. | Blinding plan, adjudication process, unblinding documentation. |  |
| 8 | Analysis populations | ITT, mITT, per-protocol, as-treated, safety, and PK/PD populations must be defined correctly. | Population definitions and subject disposition. |  |
| 9 | Multiplicity control | Controls false-positive risk across endpoints, visits, subgroups, and interim looks. | Hierarchy, gatekeeping, alpha splitting/spending, adjusted p-values. |  |
| 10 | Primary model and assumptions | Model should match endpoint type, design, and assumptions. | SAP model, diagnostics, covariance structure, PH checks if relevant. |  |
| 11 | Longitudinal or repeated measures | Visit-window, covariance, missingness, and baseline adjustment affect validity. | MMRM/GEE/mixed model details, visit definitions, sensitivity analyses. |  |
| 12 | Missing data | Amount, reasons, patterns, and assumptions can change conclusions. | Missing data tables, imputation, MNAR/tipping-point sensitivity. |  |
| 13 | Protocol deviations and adherence | Deviations, rescue medication, and adherence affect estimand and interpretation. | Deviation listings, compliance/exposure tables, rescue medication summaries. |  |
| 14 | Safety analyses | Safety conclusions require correct definitions, denominators, exposure, and event summaries. | TEAE definitions, safety population, AE tables, deaths, discontinuations. |  |
| 15 | Non-inferiority or equivalence claims | Such claims require prespecified margins, assumptions, and adequate power. | NI/equivalence margin, CI rule, assay sensitivity, constancy rationale. |  |
| 16 | Subgroup and heterogeneity claims | Subgroup effects require prespecification and interaction evidence. | Subgroup plan, interaction p-values, forest plots, exploratory labels. |  |
| 17 | Interim analyses and adaptations | Adaptations affect type I error and operational bias. | DMC charter, stopping boundaries, alpha spending, adaptation rules. |  |
| 18 | Sensitivity and robustness analyses | Tests whether conclusions depend on assumptions. | SAP sensitivity plan and sensitivity TLFs. |  |
| 19 | Effect sizes and clinical interpretation | CIs and absolute effects are necessary for clinical meaning. | Effect estimates, 95% CIs, absolute and relative effects. |  |
| 20 | Claim-language discipline | Conclusions should match design, power, multiplicity, and uncertainty. | Abstract, discussion, limitations, conclusion wording. |  |

# Checklist 3 - Observational studies

| # | Check | Why | Evidence | Checked |
|---:|---|---|---|---|
| 1 | Objective and claim type | Distinguishes descriptive, predictive, causal, safety, or prognostic claims. | Objective, estimand/target trial language, manuscript claim wording. |  |
| 2 | Protocol or analysis-plan prespecification | Reduces data-driven choices and selective reporting. | Protocol, analysis plan, registration, version date. |  |
| 3 | Study design clarity | Cohort, case-control, cross-sectional, registry, or database design determines bias risks. | Methods section and data source description. |  |
| 4 | Data-source fitness | Data provenance and completeness affect validity. | Database description, linkage quality, coding systems, capture process. |  |
| 5 | Cohort selection and index date | Incorrect index timing can create selection or immortal time bias. | Eligibility flow, index date definition, washout periods. |  |
| 6 | Exposure definition | Exposure timing, dose, duration, switching, and adherence affect interpretation. | Exposure algorithm, codes, time-varying exposure rules. |  |
| 7 | Outcome definition | Outcome validity and ascertainment affect measurement error. | Outcome algorithm, validation, adjudication, timing windows. |  |
| 8 | Confounder identification | Confounders should be clinically justified and measured before exposure. | Covariate list, DAG/rationale, baseline window. |  |
| 9 | Confounding-control method | Adjustment method must match causal question and data structure. | Regression, matching, weighting, propensity score, g-methods, IV plan. |  |
| 10 | Balance, overlap, and positivity | Poor overlap or extreme weights can make estimates unstable. | SMDs, PS distributions, weight diagnostics, effective sample size. |  |
| 11 | Time-related biases | Immortal time, time-lag, latency, and informative censoring can bias effects. | Time-at-risk definitions, follow-up rules, censoring plan. |  |
| 12 | Missing data | Missing exposure, outcome, covariate, or linkage data can bias results. | Missingness summaries, imputation plan, sensitivity analyses. |  |
| 13 | Censoring, competing risks, and recurrent events | Outcome process and follow-up must match model assumptions. | Censoring rules, competing-risk methods, recurrent event models. |  |
| 14 | Model choice and assumptions | Model should match outcome type, clustering, and follow-up. | Model specification, diagnostics, robust SEs, PH checks. |  |
| 15 | Subgroups and interactions | Subgroup findings are often exploratory and multiplicity prone. | Subgroup plan, interaction terms, CIs, multiplicity context. |  |
| 16 | Bias and sensitivity analyses | Robustness checks test alternative definitions and unmeasured confounding. | Negative controls, lag analyses, alternate definitions, quantitative bias analysis. |  |
| 17 | Effect-size reporting | Absolute and adjusted measures improve interpretation. | Adjusted risks/rates, HRs/RRs/ORs, risk differences, CIs. |  |
| 18 | Generalizability | Sample may not represent the target population. | Database population, inclusion/exclusion, baseline characteristics. |  |
| 19 | Transparency and reproducibility | Code lists, algorithms, and attrition flow enable review. | STROBE/RECORD checklist, code lists, analysis code, flow diagram. |  |
| 20 | Causal-language discipline | Observational analyses require caution about residual confounding and selection bias. | Discussion, conclusion, limitations, causal wording. |  |
